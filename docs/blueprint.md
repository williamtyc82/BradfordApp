# **App Name**: Bradford Workforce Hub

## Core Features:

- User Authentication: Secure authentication system with role-based access control (Worker/Manager) using Firebase Authentication (Google Sign-In, Email/Password).
- Training Materials Management: Centralized repository for training documents and videos, accessible to workers and managed by managers using Firebase Storage and Firestore.
- Incident Reporting System: Workers can submit incident reports with descriptions, media, and location data. Managers receive real-time notifications and can track incident status via Firestore. Generate notification emails.
- Quiz System: A platform where workers can take multiple choice quizzes and the results will be calculated and the results recorded. Managers can add more training material.
- AI-Powered Report Summarization: AI tool generates summaries of incident reports or quizzes to provide key insights to managers.
- Announcements and Notifications: Managers can create announcements to be shown to workers using the Firebase Cloud Messaging (FCM)
- Workflow Monitoring: Dashboard displaying daily activities and workflow tracking for managers to monitor worker progress.

## Style Guidelines:

- Primary color: #4A77D4 (HSL: 222, 46%, 56%), a sophisticated blue that is more vibrant than the specified brand color. 
- Background color: #F2F4F8 (HSL: 222, 30%, 95%), a very light gray-blue provides a clean, modern backdrop.
- Accent color: #59C1B8 (HSL: 169, 49%, 55%), a contrasting teal to highlight key actions and information.
- Body and headline font: 'Inter', a sans-serif font, will offer a clear, objective, and readable feel across the application.
- Use modern, outlined icons to maintain a clean and consistent look throughout the app.
- Card-based layout with ample white space to organize content effectively.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
